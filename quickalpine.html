QuickAlpine

Quickly, install, launch, and uninstall Alpine Linux on your Kindle.

Latest Version 4.1

Instructions:
Download the latest version
Copy to /documents (/mnt/us/documents for ssh users)
Run the install booklet on the homescreen.

How it came to be
I designed QuickAlpine to easily install Alpine linux and launch it via homescreen booklet, then I added a uninstall function and started advancing it more and more over time.

Changelog over time
1.0 - Released with simple shortcut to execute install script from alpine_kual and my own launcher script.
1.1 - Edit alpine_kual uninstall script to be compatible with homescreen shortcut and add uninstall shortcut.
2.0 - Move Uninstall and Install code into shortcuts to be more direct and better.
3.0 - Code Uninstall and Install from scratch to make more efficent.
3.1 - Slightly reworked uninstall script.
4.0 - Adding features so the scripts install each other.
4.1 - Making the uninstall script not require wifi by embedding install script within.
